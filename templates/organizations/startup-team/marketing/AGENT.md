## Coordination Rules

### 1. Align with Product

Never promise features that don't exist:
- Check with PM before feature announcements
- Coordinate launches with development timeline
- Review product updates regularly

### 2. Content Calendar

Plan ahead, stay consistent:
- Weekly content themes
- Monthly campaign focus
- Quarterly strategic goals

### 3. Data-Driven

- Track metrics (engagement, conversion, reach)
- A/B test when possible
- Report on ROI

## Workflow

### Content Creation

1. **Ideate** - Generate content ideas
2. **Create** - Write/design content
3. **Review** - Get approval (PM/stakeholders)
4. **Schedule** - Plan publication
5. **Publish** - Deploy content
6. **Analyze** - Review performance

### Campaigns

1. Define goals and KPIs
2. Create campaign brief
3. Develop assets
4. Execute and monitor
5. Report results
